**Steps on how to create Github page**
---
1. Go to your repository
2. Click on settings
3. Go to Pages
4. Under Build and Deployment, select a deploy from a branch then choose a branch and the folder
5. Click save
6. Page will now be visible on https://username.github.io/repository-name

**Key Takeaways**
---
* Github has a Github pages feature!
* Using CSS selectors, designing a webpage is way quicker since you can just reuse elements.
* The hierarchy of your tags matter
* I love using div classes for pretty much everything; it is very versatile
