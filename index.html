Shared Space

by MySpace

SharedSpace is a web-based art-sharing platform created using the MERN tech stack for the Young Software Engineers’ Society. It is designed for creatives to upload, showcase, and explore artworks through both private and public spaces. It includes features such as instant image sharing, a public Art Wall, voting systems, weekly challenges, leaderboards, and streak-based rewards to promote consistent engagement and creative growth. The platform welcomes community interaction, participation, and content moderation to maintain a safe, collaborative, and motivating environment for artists.

Pabili Justisya

by Skibidi Society

Pabili Justisya is a Visual Novel game created for the Los Baños Game Jam 2025 using Godot. In Pabili Justisya, you're a humble sari-sari store owner with a mission: to uncover a killer. Talk to the townspeople, do your daily work, and maybe, along the way, discover the dark truth simmering beneath the surface.

AstroMerca

by CMSC 22 Groupmates <3

AstroMerca is an e-commerce application created using Java and JavaFX. It is made to streamline interactions between buyer and seller by providing a secure and user-friendly environment. The program provides account-based access, allowing users to log in either as a Buyer or Seller to access relevant features.

Leafee

by Isha

Leafee is an optimization program created using R and R Shiny. It finds the lowest-cost combination of mitigation projects that meets the goal of reduced pollutants. It evaluates feasibility and shows the most optimal mitigation projects through an interactive interface.
